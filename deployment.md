# Google Apps Script 部署與 Sheet 設定指南

為了讓你的像素遊戲順利跑起來，請按照以下步驟完成後端設定。

## 1. 準備 Google Sheet
1. 前往 [Google Sheets](https://sheets.new) 建立一份新的試算表。
2. 更改工作表（Sheet）名稱：
   - 第一個工作表命名為：`題目`
   - 建立第二個工作表，命名為：`回答`
3. 在 `題目` 工作表填入標頭（第一列）：
   - `題號`, `題目`, `A`, `B`, `C`, `D`, `解答`
4. 在 `回答` 工作表填入標頭（第一列）：
   - `ID`, `闖關次數`, `總分`, `最高分`, `第一次通關分數`, `花了幾次通關`, `最近遊玩時間`

## 2. 貼入腳本
1. 在試算表上方選單點選 **「擴充功能」 > 「Apps Script」**。
2. 將編輯器內原本的 `function myFunction() {...}` 全部刪除。
3. 把專案目錄下的 `gas_script.gs` 內容完整複製並貼入。
4. 點選儲存（磁碟圖示），專案名稱可以取為 `PixelQuizBackend`。

## 3. 部署 Web App
1. 點選右上角的藍色按鈕 **「部署」 > 「新部署」**。
2. 選項設定：
   - **選取類型**：網頁應用程式 (Web App)
   - **說明**：v1.0
   - **執行身分**：我 (Me)
   - **誰有存取權**：任何人 (Anyone) — *這點很重要，否則前端會發生 CORS 錯誤*
3. 點選「部署」。
4. 第一次部署會跳出「要求授權」，請點選「授予存取權」並完成 Google 帳號授權。
5. 部署成功後，你會得到一個 **「網頁應用程式網址」** (以 `/exec` 結尾)。

## 4. 更新前端環境變數
1. 複製該網址。
2. 回到你的 React 專案，在根目錄建立或編輯 `.env`。
3. 貼上網址：
   ```env
   VITE_GOOGLE_APP_SCRIPT_URL=你的網址
   VITE_PASS_THRESHOLD=60
   VITE_QUESTION_COUNT=10
   ```
4. 重新啟動開發伺服器：`npm run dev`。

---

### 💡 溫馨提醒
- 每次修改 `gas_script.gs` 後，都必須點選 **「部署」 > 「管理部署」 > 「編輯」 > 「新版本」** 才能讓前端抓到更新。
- 如果遇到 CORS 錯誤，通常是因為「誰有存取權」沒有設為「任何人 (Anyone)」。
